# kuka-lwr-ros
Set of packages for simulating and controlling the KUKA Light Weight Robot (LWR). 

* **kuka_fri_bridge**  to interface and run the physical robot
  
* kuka_lwr
* kuka_planning_interface
* robot_motion_generation
* std_tools_fri
